# 简易翻译网站

一个简洁美观的翻译工具，采用蓝紫渐变色背景，支持多种语言互译。

## 功能特点

- 简洁直观的用户界面
- 蓝紫渐变色背景，白色翻译对话框
- 文本行号索引标识
- 支持多种语言互译（中、英、日、韩、法、德、西、俄等）
- 自动检测源语言
- 交换语言功能
- 文本复制功能
- 字符计数显示
- 响应式设计，支持手机和电脑访问

## 技术栈

- HTML5
- Tailwind CSS v3
- JavaScript
- Font Awesome 图标
- LibreTranslate API（免费翻译服务）

## 部署步骤

### 方法一：GitHub Pages 部署

1. Fork 本仓库
2. 在 GitHub 仓库页面，进入 Settings → Pages
3. 在 Source 选项中，选择 main 分支和根目录
4. 点击 Save，等待几分钟后即可通过 `https://你的用户名.github.io/仓库名` 访问

### 方法二：本地运行

1. 克隆本仓库：`git clone https://github.com/你的用户名/翻译网站仓库名.git`
2. 进入项目目录：`cd 翻译网站仓库名`
3. 直接打开 index.html 文件即可在浏览器中运行

## 使用说明

1. 在左侧输入框中输入需要翻译的文本
2. 选择源语言和目标语言（默认自动检测源语言，目标语言为中文）
3. 点击"翻译"按钮获取翻译结果
4. 可以使用"交换语言"按钮切换源语言和目标语言
5. 翻译结果可以通过"复制"按钮复制到剪贴板
6. "清空"按钮可以清除输入和结果

## 注意事项

- 本项目使用 LibreTranslate 免费 API，翻译质量和速度可能受限于第三方服务
- 请勿翻译过长文本，建议分段翻译
- 对于敏感内容，请谨慎使用
